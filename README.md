### 📂 Proyectos Académicos Destacados

#### 💻 Sistema de Préstamo de Equipamiento | *Desarrollo de Software (DDS)*
Desarrollo full stack de una plataforma integral para administrar solicitudes de préstamo de equipamiento, incluyendo la validación de disponibilidad, gestión de autorizaciones y registro de historial.
*   **Arquitectura:** Construido con Node.js, implementando el patrón Rutas → Controlador → Servicio → Repositorio para una clara separación entre la lógica de negocio y el acceso a datos.
*   **Seguridad:** Implementación de autenticación stateless mediante JSON Web Tokens (JWT) y encriptado de contraseñas con bcryptjs. Sistema de control de acceso basado en roles (Admin, Encargado, Usuario).
*   **Persistencia:** Modelado de base de datos relacional utilizando el ORM Sequelize con SQLite.
*   [🔗 Ver código fuente y documentación del proyecto](<./Desarrollo de sotware/TPIntegrador>)

#### ⚙️ Sistema de Gestión "Estancia del Carmen" | *Análisis de Sistemas de Información (ASI)*
Proyecto integral de análisis y diseño de software para un complejo de alojamiento turístico. El objetivo consistió en conceptualizar y diseñar una plataforma web escalable para la gestión de reservas, clientes, tarifas y servicios.
*   **Análisis de Negocio:** Relevamiento de procesos estratégicos, esenciales y de soporte. Modelado de procesos de negocio con notación BPMN.
*   **Ingeniería de Requerimientos:** Especificación de requerimientos funcionales y no funcionales (portabilidad, control de acceso, disponibilidad) y plantillas de casos de uso.
*   **Diseño de Arquitectura y UX:** Creación del Modelo de Objetos del Dominio (Diagramas de Clases). Definición de la visión del producto a través de *Buyer Personas*, Mapas de Empatía, Historias de Usuario y diseño de prototipos de interfaz.
*   [🔗 Ver documentación completa del proyecto](<./Analisis de sistemas de informacion>)

#### 🐍 Sistema de Gestión de Peajes | *Algoritmos y Estructuras de Datos (AED)*
Desarrollo backend en Python enfocado en el procesamiento lógico, análisis de registros vehiculares y gestión de patentes en estaciones de peaje.
*   **Características:** Implementación de algoritmos de búsqueda y ordenamiento, validación de formatos de cadenas de texto (patentes), y procesamiento masivo de datos.
*   **Tecnologías:** Python, estructuras de datos estáticas y dinámicas (arreglos, matrices, registros).
