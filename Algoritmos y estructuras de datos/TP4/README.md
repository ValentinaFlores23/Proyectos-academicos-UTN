# 📊 TP4: Estructuras Avanzadas y Archivos CSV

Entrega final y versión más robusta del Sistema de Gestión de Peajes. Esta versión consolida todos los requerimientos aplicando estructuras complejas de datos y algoritmos de optimización.

### 🎯 Objetivos Alcanzados
*   **Persistencia Estructurada:** Desarrollo de scripts para la lectura, parseo y procesamiento masivo de datos tabulares desde archivos `.csv` (`envios-tp4.csv`).
*   **Estructuras Avanzadas:** Implementación y manejo de vectores de registros y matrices multidimensionales para organizar la información de forma eficiente.
*   **Algoritmos de Optimización:** Aplicación de métodos de ordenamiento y algoritmos de búsqueda rápida (secuencial y binaria) para la localización y actualización de registros de patentes.
*   **Modularidad Completa:** Código estructurado estratégicamente en módulos (`modulo1tp4.py`, `modulo2tp4.py`), aislando completamente la manipulación de datos de las funciones de cálculo.
