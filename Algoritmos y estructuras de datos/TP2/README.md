# 🚘 TP2: Estructuras de Control y Validación

Segunda entrega del proyecto. En esta etapa se introdujeron las estructuras iterativas y se implementó la lógica de procesamiento de caracteres.

### 🎯 Objetivos Alcanzados
*   **Validación de Cadenas:** Desarrollo de algoritmos lógicos para validar el formato de strings (identificación y clasificación de patentes vehiculares).
*   **Procesamiento de Lotes:** Lectura y análisis de flujos de datos iniciales mediante archivos de texto de prueba (`envios25.txt`, `envios100HC.txt`, `envios100SC.txt`).
*   **Estadísticas Básicas:** Generación de métricas, conteos y porcentajes sobre el flujo de vehículos analizados.
