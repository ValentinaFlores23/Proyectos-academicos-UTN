# 🚦 TP1: Fundamentos y Lógica Secuencial

Primera iteración del Sistema de Gestión de Peajes. El objetivo de esta etapa fue establecer la lógica secuencial básica del dominio y el procesamiento matemático inicial.

### 🎯 Objetivos Alcanzados
*   **Análisis de Dominio:** Desarrollo de las primeras reglas de negocio para el cálculo y cobro de tarifas vehiculares.
*   **Control de Flujo:** Implementación de lógica condicional para clasificar tipos de vehículos y operaciones.
*   **Fundamentos Python:** Uso de variables, operadores y formateo de salidas por consola.
