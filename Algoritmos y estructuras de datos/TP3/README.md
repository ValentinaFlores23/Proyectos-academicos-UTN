# 📦 TP3: Modularización y Procesamiento de Texto

Tercera etapa del sistema. El foco principal estuvo en la reestructuración arquitectónica del código y el manejo eficiente de la memoria.

### 🎯 Objetivos Alcanzados
*   **Arquitectura Modular:** Separación de la lógica en módulos independientes (`modulo1tp3.py`, `modulo2.py`) para facilitar el mantenimiento, la legibilidad y la escalabilidad del sistema.
*   **Manejo de Archivos (I/O):** Implementación de rutinas de lectura sobre archivos de texto (`envios-tp3.txt`) para simular la entrada masiva de registros de estaciones de peaje.
*   **Estructuras de Datos:** Uso de arreglos (vectores) unidimensionales para el almacenamiento temporal y análisis de datos en memoria.
