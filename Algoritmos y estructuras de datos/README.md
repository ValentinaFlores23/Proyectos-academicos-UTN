# 🐍 Sistema de Gestión de Peajes y Patentes

**Algoritmos y Estructuras de Datos (AED) | UTN-FRC**

Este repositorio contiene el desarrollo progresivo de un sistema backend en Python para la gestión operativa y análisis de datos en estaciones de peaje. El proyecto fue construido a lo largo de cuatro Trabajos Prácticos (TPs), escalando en complejidad desde la lógica secuencial básica hasta el procesamiento masivo de información.

---

### 🛠️ Tecnologías y Conceptos Aplicados
*   **Lenguaje:** Python.
*   **Estructuras de Datos:** Arreglos (vectores), matrices y registros (objetos).
*   **Algoritmia:** Algoritmos de búsqueda (secuencial, binaria) y métodos de ordenamiento.
*   **Procesamiento de Datos:** I/O de archivos masivos, lectura y parseo de formatos `.txt` y `.csv`.
*   **Arquitectura:** Modularización del código para separar la lógica de negocio de la interfaz y los datos.

---

### 📂 Estructura del Proyecto
El sistema evolucionó en cuatro etapas. Cada directorio contiene su propia documentación específica:

*   **[`/TP1`](./TP1):** Fundamentos y lógica matemática inicial.
*   **[`/TP2`](./TP2):** Estructuras de control y motor de validación de patentes.
*   **[`/TP3`](./TP3):** Modularización arquitectónica y procesamiento de archivos de texto.
*   **[`/TP4`](./TP4):** Estructuras dinámicas complejas, búsqueda, ordenamiento y archivos CSV.
